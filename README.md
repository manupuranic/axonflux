# AxonFlux

AxonFlux is a raw-first data ingestion and analytics platform for supermarket billing software exports.

## Features
- Append-only raw ingestion
- Full auditability via import batches
- Support for item-wise & bill-wise sales and purchases
- Designed for downstream inventory, analytics, and AI enrichment

## Status
Raw ingestion layer complete for sales reports.
Purchase ingestion in progress.
